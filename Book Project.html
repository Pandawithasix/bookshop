Feather Book Website ဆိုတာဘာလဲး

Feather Book Website ဟာ ကမ္ဘာကျော်စာရေးဆရာတွေရဲ့ စာအုပ်ပေါင်း (၅၀၀)နီးပါးကို Feather Book Website မြန်မာပြန်အနှစ်ချုပ်အနေနဲ့ ဖတ်ရှုနိုင်ပြီး အချိန်နေရာမရွေး နားဆင်နိုင်မဲ့ အသံစာအုပ် (Audio books) တွေ ပါဝင်တဲ့ website တစ်ခုဖြစ်ပါတယ်။

ဘာကြောင့်စာဖတ်သင့်လဲး

တိုးတက်သောလူမျိုးသည် စာဖတ်၍ စာဖတ်သောလူမျိုးသည် တိုးတက်ကြသည်တဲ့ ကြားဖူးကြမယ်ထင်ပါတယ်။ အမှန်ပါပဲ ၊ ဘာကွာသွားလို့လဲ ဆိုတာ…..ပြောပြပေးချင်ပါတယ်။ တစ်ချို့ကတော့ ပြောတတ်ပါတယ်၊ ဘွဲ့ရတိုင်းပညာတတ်တာမဟုတ်ပါဘူးတဲ့။ မှန်ပါတယ်…..ပညာဆိုတာ အမှားအမှန်ကို ပိုင်းခြားသိမြင်နိုင်တဲ့ဉာဏ်ကို ပြောတာပါ။ ဘယ်ကဘယ်လိုရထားပါတယ်ဆိုတဲ့ ဘွဲ့လက်မှတ်တွေကို ပြောတာမဟုတ်ပါဘူး။ အတန်းပညာမှာတော်တတ်လွန်းလို့ ဘွဲ့လက်မှတ်တွေ တသီတတန်းကြီးရတိုင်း ပညာတတ်တာမဟုတ်ပါဘူး။ ထိုဘွဲ့လက်မှတ်တွေက ကိုယ့်ရဲ့အရည်အချင်းတစ်ခုကို သတ်မှတ်ပေးထားတဲ့ အရေအတွက်တစ်ခုပါ။ ထိုအရေအတွက်တွေအားဖြင့်တော့ အကြမ်းဖျင်း သတ်မှတ်ပြီး အလုပ်ခွင်မှာ ရာထူးနေရာကွဲပြားတာ၊ ဝင်ငွေကွဲပြားတာတွေ ဖြစ်လာရတာပါ။

စာဖတ်ခြင်းအကျိုးကျေးဇူး

စာတွေကိုအမြဲတမ်းဖတ်တဲ့သူတွေဟာ သာမန်သူတွေထက်စာရင် ပိုမိုပြီးကျယ်ပြန်တဲ့ စိတ်ကူးတွေ ၊ ဖန်တီးနိုင်စွမ်းတွေ မြင့်မားတတ်ကြပါတယ်။ စာဖတ်တဲ့အခါမှာ ကိုယ်ဖတ်နေတဲ့ စာအုပ်အပေါ်ကို မူတည်ပြီးတော့ စိတ်ကူးထဲမှာ ပုံဖော်တတ်ခြင်းဟာ စာအုပ်ဖတ်တဲ့သူတိုင်း လုပ်တတ်ကြတဲ့ အမူအကျင့်တစ်ခုပါပဲ။ စာအုပ်တွေကို အမြဲတမ်းဖတ်တဲ့သူတွေဟာ အမြဲတမ်းလိုလို စိတ်ကူးယဉ်ပြီး စိတ်ကူးတွေပုံဖော်လေ့ရှိကြတဲ့ အမူအကျင့်တွေကြောင့် သာမန်သူတွေထက်စာရင် ပိုပြီး ကျယ်ပြန်တဲ့ အတွေးအခေါ်တွေ ၊ ဖန်တီးနိုင်စွမ်းတွေဟာ အမြဲတမ်းတိုးတက်စေလာနိုင်တာပဲဖြစ်ပါတယ်။
